# Règles permanentes pour ce projet (Réparto)

## Git / GitHub
- Ne JAMAIS faire de commit ou de push vers GitHub sans mon autorisation explicite.
- Toujours me demander avant d'envoyer du code sur le dépôt distant.

## Argent et sécurité (projet fintech)
- Ce projet déplace de l'argent réel. Ne jamais toucher à la logique de paiement, 
  de payout ou de commission sans me prévenir et m'expliquer d'abord.
- Ne jamais exposer une clé API en clair (ni en base, ni côté navigateur).
- Toujours tester en sandbox avant tout code lié au vrai argent.

## Base de données
- Ne jamais supprimer une table ou des données sans mon autorisation explicite.
- Toujours respecter la sécurité RLS (chaque utilisateur ne voit que ses données).
